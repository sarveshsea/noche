dibs Feature Set

Complete feature breakdown for AICP-accurate production bidding. 18 feature domains, organized by surface and priority, with every input mapped to a stepper step.

Design Process
1
Recognized the problem
An AICP bid is fundamentally a long form with hundreds of inputs across sections A through E. Spreadsheets force users to manage all of them at once.
2
Broke it down
Identified required vs optional inputs per AICP section. Grouped by dependency: project metadata drives shoot params, which drive crew, which drive budget.
3
Distributed into steps
Four stepper stages that progressively build the bid. Each step only asks for what it needs, prefills what it can, and validates before moving forward.

AICP Stepper Breakdown

01
Type
Choose bid type or start from a template. Templates prefill shoot days, crew size, locations, union status, and usage defaults.
Template Selection
Bid type (TV, Digital, Integrated, Photo, Animation, Documentary)
Quick Config
Shoot days, crew size, locations, format, union, usage territory
02
Inputs
Project details, shoot parameters, usage rights, and financial globals. These drive downstream calculations across all budget sections.
Project Details
Project name, client, agency, producer, bid date, job number Required
Shoot Parameters
Shoot days, pre-pro days, strike days, format, locations, crew size Required
Usage & Rights
Media type, usage period, territories, digital extension, exclusivity
Financials
Union type, payroll fringes, OT multiplier, markup %, contingency %, agency commission %, handling fee % Required
03
Budget
AICP sections A through E with full line-item detail. Each section has categories with individual cost entries, rates, overtime, fringes, and allowances.
AICP Sections
A. Pre-Production / B. Production / C. Post-Production / D. Insurance / E. Agency Fees
Per Line Item
Description, category, unit type, qty, rate, OT, fringes, allowance, notes
Conditional Fields
Talent usage (session/buyout), equipment rental logic, travel cost logic, contingency, markup stacking
04
Review
Totals, cost breakdown, AI insights, validation, and export. Final confirmation before submission with approval workflow.
Summaries
Grand total, section subtotals, top cost drivers, markup/fee breakdown, budget distribution
Validation
Required field checks, inline validation, submit readiness score, error summary with jump-to-row
Export
Clean AICP form, PDF, Excel, CSV, print-friendly layout

Full Feature Spec (18 Domains)

  • Edit existing bid
  • Duplicate bid
  • Save as new version
  • Project metadata entry
  • project name, client, agency, production company, bid date, version number
  • Status management (draft, in review, submitted, approved/locked)

  • Archive bid
  • Reopen archived bid
